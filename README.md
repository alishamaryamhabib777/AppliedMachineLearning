## 🏠 Week 1: Real Estate Regression Analysis (Detailed)

### 📋 Objective
Predict residential house prices in India using key housing attributes. This task focuses on conducting Exploratory Data Analysis (EDA), handling missing data, normalizing features, training baseline and ensemble regressors, and evaluating performance.

### 📊 Dataset
* **Source:** [House Price India Dataset (Kaggle)](https://www.kaggle.com/datasets/mohamedafsal007/house-price-dataset-of-india)
* **Features:** Quantitative and qualitative details of residential properties.

### 🛠️ Key Steps Implemented
1. **Exploratory Data Analysis (EDA):**
   * Plotted house price distributions to check skewness.
   * Generated a correlation heatmap to identify relationships between the independent variables and the target price.
2. **Data Preprocessing:**
   * Handled missing values by replacing them with column medians.
   * Split the dataset into a **80% training set** and a **20% testing set**.
   * Normalized numerical features using `StandardScaler`.
3. **Machine Learning Modeling:**
   * Trained a baseline **Linear Regression** model.
   * Trained an ensemble **Random Forest Regressor** (100 estimators).
4. **Evaluation:**
   * Computed Mean Absolute Error (MAE), Root Mean Squared Error (RMSE) and $R^2$ Score to compare model performance.
